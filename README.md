# Relogio-Digital
Um relógio digital super simples, que mostra a hora atual do seu Pc.

  Windows:

    Instalar o Python, ativando a caixinha "Add Python to PATH"

    O Tkinter já vem instalado no Windows!
  
  Linux:
    Instalar o Python e o pip pelo comando "sudo apt-get install python3-pip"

    1º Jeito:
    
      sudo apt install python3-pip python3-tk
      
  Mac OS:

    Instalar o Python pelo comando: "brew install python"
    
    O Tkinter já vem instalado no Mac OS!
